What is the difference between map(), filter(), and forEach()?
map() → transforms each element → returns a new array

filter() → keeps elements that match a condition → returns a new array

forEach() → runs a function on each element → does NOT return anything
 What is an arrow function?

 What are template literals?

Template literals are a way to create strings in JavaScript using backticks ` instead of quotes.
